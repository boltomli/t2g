## 功能特性

### 1. 文本分析
- 使用LLM分析文本内容
- 提取世界观、角色、冲突、主题
- 生成结构化的分析结果

### 2. 游戏类型推荐
- 基于文本特征推荐3-5种游戏类型
- 支持RPG、冒险、视觉小说、策略、动作
- 提供详细的推荐理由和特性说明

### 3. 游戏生成
- 自动生成完整的Godot项目
- 包含场景、脚本、资源
- 支持多种游戏类型

### 4. 双模式运行
- **Godot应用模式**: 独立的图形界面应用
- **Pi Agent模式**: 在编程agent中使用

## 使用方法

### 快速开始
1. 启动LM Studio并加载模型
2. 运行`launch.bat`或使用Godot打开项目
3. 输入文本并点击分析
4. 选择游戏类型
5. 等待游戏生成完成

### 详细说明
请参考`QUICKSTART.md`
