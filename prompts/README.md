# 提示词文件

这些提示词用于指导本地LLM进行文本分析。

## 文件说明

| 文件 | 用途 | 说明 |
|------|------|------|
| analyze.txt | 文本分析 | 提取世界观、角色、冲突等信息 |
| recommend.txt | 类型推荐 | 基于分析推荐游戏类型 |
| merge.txt | 结果合并 | 合并多个分块分析结果 |

## 设计原则

1. **简洁明了**: 适合小模型理解
2. **结构清晰**: 明确的输出格式要求
3. **少即是多**: 避免过长的提示词

## 自定义

修改对应的 .txt 文件即可自定义提示词。

注意：保持 JSON 输出格式不变。
